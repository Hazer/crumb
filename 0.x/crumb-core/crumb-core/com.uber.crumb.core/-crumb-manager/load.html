fun <T : Serializable> load(nameFilter: (String) -> Boolean): Set<T>

This loads a given Set<T> from the Crumb META-INF store that matches the given nameFilter.

Parameters

nameFilter - a name filter to match on. Conventionally, one could use a "known" file extension used for file names in store.

Return
the loaded Set<T>, or an empty set if none were found.
